Estruturas de decisão

Condição

Ao escrever um algoritmo nos deparamos com a necessidade de dizer qual caminho o program deve tomar (uma decisão) para cada situação apresentada.
E é exatamente para isto que as "estruturas de decisão" servem.
A estrutura mais utilizada é a se-então-senão - em inglês If-Then-Else.
O funcionamento é simples, com base no resultado de uma expressão onde a resposta deve ser VERDADEIRO ou FALSO (também conhecida como expressão boleana), o fluxo do algoritmo segue para um bloco de instruções ou não.

Estrutura de Repetição

While e For

A estrutura de repetição permite que um conjunto de instruções seja executado repetidamente até que ocorra uma condição de parada.
Evita que o mesmo conjunto de instruções seja escrito várias vezes.
